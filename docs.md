## 1. install go 
```bash
sudo apt update
```

```bash
wget https://go.dev/dl/go1.20.7.linux-amd64.tar.gz
```

```bash
sudo tar -C /usr/local -xzf go1.20.7.linux-amd64.tar.gz
```

```bash
export PATH=$PATH:/usr/local/go/bin
```

```bash
source ~/.profile 
```

```bash
go version
```

```bash
go mod init main
```

```bash
go run .
```
